# API de Produtos

API REST desenvolvida em Java com Spring Boot para gerenciamento de produtos.

## Tecnologias utilizadas
- Java 25
- Spring Boot
- PostgreSQL
- Maven

## Estrutura do projeto
- `entities` – Entidades do domínio
- `enums` – Enumerações do sistema
- `factories` – Fábrica de conexões com o banco de dados
- `sql` – Scripts de criação do banco
- `api` – Classe principal da aplicação

## Banco de dados
- PostgreSQL
- Script de criação da tabela disponível em `/sql`
